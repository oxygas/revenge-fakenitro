# FakeNitro for Revenge

A port of Vencord's FakeNitro plugin for [Revenge](https://github.com/revenge-mod/revenge-bundle).

Allows you to send fake emojis/stickers, use nitro themes, and stream in nitro quality without having Nitro.

## Installation

Paste this URL into Revenge's plugin section:

```
https://oxygas.github.io/revenge-fakenitro/fakenitro
```
